# CNRoutingAssignment

**Module:** Computer Networks
**Student ID:** 39332

## Assignment Description
This repository contains the Cisco Packet Tracer (.pkt) files for the Router Configuration and Routing Protocols practical assignment. It demonstrates basic router setup, static routing concepts, and dynamic routing implementations using both RIP and EIGRP.

## Repository Contents
* **Task 1.pkt**: Contains the basic router configurations, including the custom hostname setup and console password authentication.
* **Task 2.pkt**: Contains the physical network topology, IP address configurations, and the static routing setup.
* **Task 3.pkt**: Contains the dynamic routing implementations, covering the configurations for both Part A (RIP routing) and Part B (EIGRP routing).
